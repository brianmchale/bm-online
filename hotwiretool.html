Hot Wire Tool

The purpose of this project was to create a handheld hot wire tool that could be used in a variety of applications. As a group, we discussed numerous different ideas as to what form this tool could take. We chose this design because it made the most sense ergonomically for what we were trying to use the tool for, and it allowed for interchangeable nichrome wiring via built-in alligator clips.
